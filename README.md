## Discord Rich Presence - QBcore


## Japanese(日本語)
## 特徴

- QBCoreを元に作られています
- Disocrdのアクティビティに サーバー名, サーバー人数, 起動時間 が表示されます
- ボタンが二つありサーバーに参加できるボタンやDiscordに参加できるボタンが設定できます
